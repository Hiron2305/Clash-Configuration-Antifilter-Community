# Advanced Mihomo/Clash Configuration

Репозиторий содержит оптимизированную конфигурацию для ядра **Mihomo (Clash Meta)**, адаптированную для использования в среде **Clash Verge Rev**. 

## Технические особенности
- **Smart Routing Logic:** Реализована гибридная модель маршрутизации на основе черных списков (Blacklist-first). Трафик по умолчанию направляется через локальный шлюз (DIRECT), в туннель уходят только заблокированные ресурсы и сервисы с геоблокировками.
- **Process-Level Isolation:** Использование правила `PROCESS-NAME` для изоляции трафика конкретных приложений (например, Discord), что исключает необходимость проброса широких диапазонов портов.
- **CDN Optimization:** Выделенные правила для контентных сетей (Steam CDN, Microsoft/Xbox, Epic Games), гарантирующие максимальную пропускную способность и минимальные задержки.
- **Protocol Hardening:** Блокировка протокола QUIC (UDP:443) для принудительного перехода на TCP внутри туннеля, что повышает стабильность соединения при работе через VLESS-Reality.
- **Automated Rule Providers:** Динамическая подгрузка актуальных списков заблокированных доменов (Antifilter Community) с интервалом обновления 12 часов.
- **DNS Configuration:** Настроенный Fake-IP стек для корректного резолвинга доменов в TUN-режиме.

## Развертывание
1. Установите актуальную версию [Clash Verge Rev](https://github.com/clash-verge-rev/clash-verge-rev).
2. Скопируйте `vpn.yaml` в рабочую директорию профилей.
3. Отредактируйте блок `proxies`, указав параметры вашего VLESS-сервера (IP, UUID, SNI, Public Key).
4. Импортируйте профиль в GUI.
5. **Важно:** Для корректной работы системы перехвата пакетов необходимо установить `Service Mode` и активировать `TUN Mode` в настройках приложения.

## Спецификация маршрутов
- **Proxy:** Социальные сети, медиа-сервисы (YT, IG, TW), Discord, Steam Cloud/Auth, базы Antifilter.
- **Direct:** Весь RU-трафик, игровые серверы, локальные сервисы, тяжелые загрузки контента.
